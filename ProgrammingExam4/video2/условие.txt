! package не использовать
Во всех входных файлах каждая запись находится на новой строке.
Поля одной записи отделяются друг от друга символом точка с запятой.
В конце строки точка с запятой НЕ ставится.
!!! В выходных файлах не должно быть дополнительных пустых строк в конце.
Примеры входных и выходных файлов вы можете посмотреть на вкладке Tests / Тест кейсы для данной задачи.

Во входном файле input1.txt находится информация о сотрудниках.
Каждый сотрудник описан с новой строки: id, фамилия и имя сотрудника, дата начала работы, дата окончания контракта, количество оставшихся дней отпуска, отгулов. Формат даты: dd.mm.yyyy
empl_id;empl_surname;empl_name;start_date;end_date;vacation_days;compensatory_days

Во входном файле input2.txt находятся запросы от сотрудников. Каждый запрос с новой строки.
request_id;empl_id;daysoff_id;days
Где daysoff_id = 1 для отпуска, daysoff_id = 2 для отгулов.

2 балла. В выходной файл output1.txt вывести все некорректные запросы сотрудников на отпуск (vacation, количество запрошенных дней больше, чем осталось дней в input1.txt).
Вывести запросы в порядке их следования во входном файле input2.txt. Каждый запрос с новой строки
request_id;empl_id;daysoff_id;days

2 балла. В выходной файл output2.txt вывести по заданному количеству дней (файл input3.txt) список всех сотрудников,
у которых количество дней отпуска (vacation) в input1.txt >= указанного количества дней, по алфавиту фамилий.
empl_surname;empl_name;vacation_days

2 балла. В выходной файл output3.txt вывести список сотрудников по убыванию количества отработанных в компании дней (days) до окончания контракта.
Для одинакового количества дней вывести по алфавиту фамилий. Каждую запись вывести с новой строки.
empl_surname;empl_name;days

4 балла. В выходной файл output4.txt вывести по заданному empl_id (входной файл input4.txt) сколько дней отпуска необходимо компенсировать сотруднику в случае его увольнения сегодня (текущая дата).
Расчет количества дней следующий: за каждый полный отработанный месяц от сегодня до конца контракта сотруднику доступно 2 дня отпуска (vacation).
Количество дней для компенсации = количество оставшихся дней отпуска из файла input1.txt минус количество положенных дней отпуска от сегодня до конца контракта.
Вывести число, если оно больше или равно 0. Если сотрудника с заданным id нет, то вывести минус 1.
